# KAIST Case Hub

KAIST MBA Spring 2026 케이스 스터디 허브입니다. AI, 벤처캐피탈 등 주요 비즈니스 주제별로 케이스 자료를 정리하고 제공합니다.

## 구성

| 디렉토리 | 내용 |
|---|---|
| `ai-evolution/` | AI 발전과 비즈니스 전략 케이스 |
| `venture-capital/` | 벤처캐피탈 투자 및 전략 케이스 |
| `tools/` | 케이스 분석 지원 도구 |

## 사용법

`index.html`을 브라우저에서 열거나, GitHub Pages를 통해 웹에서 접근합니다.

## 기술 스택

- HTML / CSS (정적 사이트)
- Python (데이터 처리)
